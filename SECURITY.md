# Chính sách bảo mật.

## Các phiên bản được hỗ trợ


| Version | Supported          |
| ------- | ------------------ |
| Test 0.0.0   | :x: |
| Stable v0.1.0   | :x: |
| Final 1.0.0   | :white_check_mark: |

## Báo cáo lỗi:

Báo lỗi bảo mật bằng cách tạo issue kèm theo tag 'bug'.
Các lỗi bảo mật sẽ được kiểm tra hàng tuần
